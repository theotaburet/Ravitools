# Skill: GPX Validation

## Quand utiliser ce skill

Utiliser ce workflow pour verifier qu'un changement reste coherent sur un vrai trajet.

## Checklist

1. Le GPX est parse sans erreur.
2. Le lissage ne detruit pas la forme utile de la trace.
3. La requete OSM reste raisonnable.
4. Les POI resultants sont proches de l'itineraire.
5. Le bruit reste acceptable.
6. L'export final reste exploitable.
