# MemPalace Notes

## Setup local

- Python compatible: `python3.12`
- Venv: `./.tools/mempalace-py312`
- Commande: `./.tools/mempalace-py312/bin/mempalace`
- Palace: `./.mempalace/palace`

## Usage conseille

- `status` pour verifier l'etat de la memoire.
- `search` pour retrouver une decision ou un fichier deja explore.
- `wake-up` pour un rappel compact du contexte memoire.

## Limite connue

- Le package casse avec Python 3.14 dans cet environnement.
